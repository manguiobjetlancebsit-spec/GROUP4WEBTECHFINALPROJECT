 Daily Brew
Your Daily Dose of Coffee

A coffee shop website built as a group web development project for BSIT 1-P, Group 4.

 Project Goals
Build a fully functional and responsive coffee shop website
Practice real-world web development using HTML, CSS, and JavaScript
Apply UI/UX design principles to create an engaging user experience

 How to Run Locally
Clone the repository
   git clone https://github.com/manguiobjetlancebsit-spec/GROUP4WEBTECHFINALPROJECT/tree/main
Navigate to the project folder
   cd daily-brew
Open the project
Open index.html in your browser, or
Use the Live Server extension in VS Code for best experience
No installations or dependencies required. Pure HTML, CSS, and JS!
